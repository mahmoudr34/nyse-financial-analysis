# Financial Performance Analysis of NYSE-Listed Companies

## Overview
This project analyzes financial data from NYSE-listed companies to evaluate revenue growth, profitability, and performance trends using Excel.

## Business Objectives
- Analyze revenue growth over time
- Compare profit margins across sectors
- Identify high-performing companies
- Support data-driven decision-making

## Tools Used
- Excel 
- GitHub 

## Key Insights
- Revenue growth is concentrated among a small number of companies
- High growth does not always indicate strong profitability
- Profit margins vary significantly by sector

## Deliverables
- Excel dashboard: `/excel/NYSE_Financial_Analysis.xlsx`
- PDF report: `/report/NYSE_Financial_Analysis.pdf`
- Dashboard visuals: `/visuals/`

## Dataset
Source: Kaggle (NYSE Financial Statements)
